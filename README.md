# 🏠 California Housing Price Prediction

This project implements a Machine Learning regression model to predict housing prices using the California Housing dataset.

## 📊 Dataset

* Source: Scikit-learn built-in dataset
* Features include:

  * Median Income
  * House Age
  * Average Rooms
  * Population
  * Location (Latitude & Longitude)

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 🚀 Workflow

1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Feature Scaling using StandardScaler
5. Model Training (Linear Regression)
6. Cross Validation
7. Prediction

## 📈 Model Used

* Linear Regression

## 📌 Results

* Model evaluated using Mean Squared Error
* Cross-validation applied for better performance estimation

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

* Predicted housing prices based on test data
